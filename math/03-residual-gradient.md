# 03｜ResNet 残差连接的梯度推导

## 1. 残差块

忽略激活时，残差块输出：

```text
y = F(x;W) + x
```

其中 F 是卷积支路，x 是恒等捷径。

## 2. 对输入的梯度

```text
∂L/∂x = ∂L/∂y · (∂F/∂x + I)
       = ∂L/∂y · ∂F/∂x + ∂L/∂y
```

第二项提供不经过卷积权重的直接梯度通路。即使 `∂F/∂x` 很小，仍可保留上游梯度。它缓解优化困难，但不能从数学上保证所有网络永不梯度消失。

## 3. 对残差参数的梯度

捷径不依赖 W：

```text
∂L/∂W = ∂L/∂y · ∂F(x;W)/∂W
```

参数仍通过残差支路正常学习。

## 4. ReLU 后激活

本仓库使用 `y=ReLU(F(x)+S(x))`。令 `a=F(x)+S(x)`：

```text
∂L/∂a = ∂L/∂y ⊙ 1[a>0]
∂L/∂x = ∂L/∂a · (∂F/∂x + ∂S/∂x)
```

若 S 是恒等映射，`∂S/∂x=I`；若用 1×1 卷积对齐，S 也有参数并接收梯度。

## 5. 为什么需要 1×1 shortcut

主支路若从 `[B,16,28,28]` 变成 `[B,32,14,14]`，不能直接与 x 相加。`Conv1x1(stride=2)` 将通道与空间尺寸对齐，其权重也参与反向传播。

## 6. 多个残差块

若 `x_{l+1}=x_l+F_l(x_l)`，展开得到后层表示含前层的加法通路。反向梯度也包含多个恒等与残差路径之和，这是 ResNet 易于优化的核心。
